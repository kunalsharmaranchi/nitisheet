# Niti Sheet — Spreadsheet with Nested Sheets

**Repository description:** Niti Sheet is an offline desktop spreadsheet application with Nested Sheets: any cell can contain its own child spreadsheet, structured schema, validation rules, formulas, forms and data tools.

Niti Sheet is designed for users who need more than a flat spreadsheet. Alongside normal cells, formulas, charts and reports, it lets a cell open a complete child spreadsheet called a **Nested Niti Sheet**.

## The key feature: Nested Sheets

With a normal spreadsheet, detailed information usually requires many extra rows, columns or separate files. Niti Sheet keeps the parent sheet clean:

- Right-click a cell and choose **Niti Sheet → Create Niti Sheet**.
- That cell becomes a container for its own child spreadsheet.
- Double-click an existing Nested Sheet to open it.
- Create child schemas with their own fields, validation and nested levels.
- Use nested paths in formulas, such as `A2#B6#H6`.
- Add Master Shell formulas to a nested schema so new child sheets receive the formula automatically.

This is useful for invoices and line items, customer records, student details, projects, inventory, case files and any data where one main record needs deeper details.

## Main capabilities

- Offline spreadsheet workbooks, formulas, formatting, charts, images and PDF/print output
- Nested Niti Sheets inside cells
- Nested schema designer, validation rules and form-based data entry
- Master Shell formulas for column summaries
- CSV and JSON import/export
- Sorting, filtering, conditional high/low highlighting and Batch Edit
- Data-quality dashboard and schema checking
- Intent / FAQ chatbot workspace with Package, Embed, Widget, Client Bundle and Hosted Config exports
- Autosave and recovery after an unexpected close

## User manual

Open [index.html](index.html) in a web browser. Keep the `assets` folder beside the manual because it contains all feature screenshots.

## Important feature paths

- **Create Nested Sheet:** Right-click cell → Niti Sheet → Create Niti Sheet
- **Create Nested Schema:** Data → Quality → Designer → Input Rules → Nested Sheet → Design Nested Schema
- **Add Nested Master Formula:** Design Nested Schema → Add Master Formula
- **Create Chatbot Intents:** Data → Chatbot → Create
- **Batch Cleanup:** Data → Batch Edit → Batch
