Niti Sheet — Button-by-Button Guide

Read this alongside the illustrated complete manual.

Why it is unique: Niti Sheet combines normal spreadsheet work with Nested Cell Sheets, schema-to-form design, data-quality scoring, and a Chatbot Create / Intent FAQ Workspace in one offline desktop program.

File

ButtonPurpose
NewCreates a new blank workbook.
OpenOpens an existing saved workbook from your computer.
SaveSaves the current workbook, including sheets, formulas and formatting.
CSV ImportLoads a CSV table into the workbook.
CSV ExportSaves simple sheet data as CSV for another system.

Home

ButtonPurpose
Paste / Copy / CutPaste copied cells, copy without changing originals, or move cells via cut then paste.
Undo / RedoReverse the last action or restore an undone action.
Font / sizeSets text font and point size for selected cells.
Bold / ItalicChanges text emphasis.
Fill / TextSets cell background and text colour; palettes include reset choices.
Left / Center / RightHorizontally aligns selected cell contents.
WrapDisplays long text on more than one line in its cell.
Auto SizeResizes selected cells to better fit their contents.
LayoutOpens Merge Cells, Unmerge Cells, Wrap Text, Auto Size Cells, AutoFit Columns and AutoFit Rows.

Insert

ButtonPurpose
New / Rename / DeleteAdds, renames or deletes the active worksheet.
Nested Open / CloseOpens a mini child sheet associated with the selected cell, or returns to the parent sheet.
Chart / InsertDisplays a chart view or creates a chart from selected prepared data.
Chart Copy / Cut / Paste / Edit / DeleteManages a selected chart object.
Image / DeleteImports an image at the selected location or removes the selected image.
TableFormats the selected range as a clear table.

Data

ButtonPurpose
CSV Import / ExportImports or exports simple CSV tables.
Chatbot CreateOpens structured-data import/export, testing, package and integration tools.
CreateOpens Intent / FAQ Workspace and creates structured question/answer records.
BatchRuns a controlled bulk-edit operation on selected data.
Sort A–Z / Z–ASorts the selected column ascending or descending.
Filter / ClearShows only rows matching a condition, or restores every row.
DashboardOpens quality indicators for completeness, validation, duplicate checks and schema health.
DesignerOpens Schema-to-Form Designer for form-driven data entry.
Validate / Check / ClearSets validation, checks configured rules/schema, or removes validation.
High/LowHighlights high or low values conditionally.

Review

ButtonPurpose
FindLocates a text string or value.
ReplaceFinds matching content and substitutes a new value; review the target before applying broadly.

View

ButtonPurpose
Show FormOpens the active sheet as a data-entry form.
Master Shell: ShowShows or hides the Master Shell row.
Master Shell: FormulaSets the formula associated with the Master Shell row.
Zoom Out / Reset / Zoom InChanges the visual scale of the worksheet; Reset returns to 100%.
Grid Lines: Softer / StrongerReduces or increases grid-line visibility without changing data.
ThemeChooses a Light Theme or Soft Dark Theme; only the appearance changes.
SetupSets paper, orientation, dimensions, margins and print range.
PrintPrints the workbook through the printer workflow.
PDFCreates a PDF copy of the workbook.
Show Area / Hide AreaShows or hides the print-area boundary on the worksheet.
